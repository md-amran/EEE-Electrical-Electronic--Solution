EE Hub-এর সকল ফিচার ব্যবহার করার জন্য প্রথমে আপনাকে অ্যাকাউন্ট তৈরি করতে হবে:

  1. হোমপেজের উপরের ডান কোণায় 'সাইন আপ' বাটনে ক্লিক করুন

  5. লগইন করুন এবং ড্যাশবোর্ড এক্সেস করুন

উদাহরণ Python কোড 

# LED ব্লিংক সিমুলেশন
import time

def blink_led(pin, times, delay):
    for i in range(times):
        print(f"LED {pin} ON")
        time.sleep(delay)
        print(f"LED {pin} OFF")
        time.sleep(delay)

# ব্যবহার
blink_led(13, 5, 0.5)
print("ব্লিংকিং সম্পন্ন!")

Arduino কোড আপলোড করা

Arduino কোড সরাসরি ডিভাইসে আপলোড করতে:

  1. Arduino বোর্ড সিলেক্ট করুন (UNO, Nano, ESP32, ইত্যাদি)
  2. কোড লিখুন এবং কম্পাইল করুন
  3. Web Serial API ব্যবহার করে কানেক্ট করুন
  4. আপলোড বাটনে ক্লিক করুন
  5. সিরিয়াল মনিটরে আউটপুট দেখুন
মনোযোগ: Web Serial API শুধুমাত্র Chrome/Edge ব্রাউজারে কাজ করে। অন্যান্য ব্রাউজারের জন্য আপনাকে Serial Driver ইনস্টল করতে হতে পারে।

Arduino MQTT কোড উদাহরণ 

#include 
#include 

// WiFi credentials
const char* ssid = "your_SSID";
const char* password = "your_PASSWORD";

// MQTT Broker
const char* mqtt_server = "test.mosquitto.org";
const int mqtt_port = 1883;

WiFiClient espClient;
PubSubClient client(espClient);

void setup() {
  Serial.begin(115200);
  
  // WiFi কানেক্ট
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  
  // MQTT কানেক্ট
  client.setServer(mqtt_server, mqtt_port);
  
  if (client.connect("ESP32Client")) {
    client.publish("eehub/device/temperature", "25.5");
    client.subscribe("eehub/device/control");
  }
}

void loop() {
  client.loop();
  
  // সেন্সর ডাটা পাঠান
  float temp = readTemperature();
  char tempStr[8];
  dtostrf(temp, 4, 2, tempStr);
  client.publish("eehub/device/temperature", tempStr);
  
  delay(5000);
}
